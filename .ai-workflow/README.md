# AI Workflow Documentation

This directory contains all artifacts from the AI-powered development workflow.

## Structure

- **context/** - System and product context, feature registry
- **specs/** - All specifications (ideas, features, enhancements, reviews)
- **design/** - UI/UX designs and style guide
- **architecture/** - Technical architecture documents and proposals
- **plans/** - Implementation plans and roadmaps
- **implementation/** - Progress tracking during development
- **knowledge/** - Learnings, patterns, and optimizations

## Workflow

See [AI-DEVELOPMENT-WORKFLOW.md](./AI-DEVELOPMENT-WORKFLOW.md) for the complete workflow guide.

## Commands

Use the slash commands in Claude Code to navigate the workflow:
- `/spec-init` - Initialize context (first time only)
- `/idea` - Review and enhance ideas
- `/create-spec` - Create specifications
- `/design` - Design UI/UX
- `/engineering-review` - Technical review
- `/architect` - System architecture
- `/plan` - Implementation planning
- `/implement` - Execute the plan
- `/update-spec` - Document final state
